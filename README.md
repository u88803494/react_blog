## React SPA BLOG
此為本人 React 練習 project，環境使用 create-react-app 建置。

## 簡介
主要功能如下：
- 首頁：呼籲來寫文章

- 文章列表頁面：顯示文章以及新增、編輯、刪除文章的界面。

- 會員系統：支持 Google 登入。

- 利用 Cookie 來維持登入狀態。

- 文章列表支持以兩種形式顯示文章列表，一種是顯示部份文章內容，另外一種是條列式。

- 當登入之後才可以使用新增、刪除、修改文章的功能，修改、刪除功能顯示在條列式。

- 可以用 markdown 撰寫文章。

- 關於我頁面：目前有一篇內容，之後要改成自己的介紹


## 使用技術
- React
- React hook
- Redux
- React-Router
- Redux-thunk
